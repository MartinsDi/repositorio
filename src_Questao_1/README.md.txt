Iniciar a aplicação

Aparecera uma janela solicitando a quantidade de dígitos que será utilizado, caso digite algum caractere diferente de um numero irá aparecer uma janela informando o erro e pedindo para redigitar.

Em seguida uma nova janela aparecerá e terá que digitar cada número por vez e confirmando cada solicitação e caso for digitado algo diferente de número será pedido para redigitar.

Ao terminar de digitar todos os números aparecerá uma janela mostrando o vetor com todos os elementos e a mediana.
